# **Weekly Report Generation Prompt**

## **1. Objective**

Take the provided list of research findings/mentions and generate a **Weekly Media & Mentions Briefing Note**.

The output must be professional, factual, and formatted for direct email distribution.

## **2. Input Data**

[PASTE THE RESEARCH FINDINGS HERE]

## **3. Processing Instructions**

1.  **Filter & Group:** Group multiple sources covering the same event under a single "Issue" heading. Discard irrelevant or duplicate noise.
2.  **Identify Triggers:** Clearly state the "Trigger" (the core event/announcement) for each issue.
3.  **Report on Dissent:** If the research includes opposing views, ensure these are highlighted to show the full breadth of the debate.
4.  **Tone:** Strictly factual. No editorializing.
5.  **Recency:** Ensure all items fall within the reporting period (last week).
